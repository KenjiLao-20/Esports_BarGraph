# 📊 Game Release Year Analysis

<br>

A data visualization tool that analyzes and displays the distribution of video games by their release year from CSV data.

<br>
<br>

## ✨ Features

<br>

- **CSV Upload** – Upload any game dataset CSV file
- **Bar Chart Visualization** – Interactive Chart.js bar chart
- **Statistics Dashboard** – Shows total, average, min, and max release years
- **Dark/Light Mode Toggle** – Theme preference with smooth transition
- **Data Export** – Download processed data as CSV

<br>
<br>

## 📈 Statistics Displayed

<br>

| Statistic | Description |
|-----------|-------------|
| Total Games Released | Sum of all games in dataset |
| Average Games Per Year | Mean distribution across years |
| Year with Most Releases | Peak release year |
| Year with Least Releases | Lowest release year |

<br>
<br>
