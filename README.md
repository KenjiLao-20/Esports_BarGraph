## 🚀 How to Use

<br>

1. Open `index.html` in your browser
2. Click "Choose File" and select your CSV
3. Click **"Process Data"** to generate the chart
4. Click **"Export Data"** to save results as CSV

<br>
<br>

## 🎨 Theme Toggle

<br>

Use the 🌞/🌙 switch in the bottom-left corner to toggle between light and dark mode.
